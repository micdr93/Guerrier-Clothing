## User Story Testing

Each user story has been tested to ensure the acceptance criteria are met. Here's a summary of the testing conducted for key user stories:

### Authentication & User Profiles

| User Story | Test Procedure | Result | Pass/Fail |
|------------|---------------|--------|-----------|
| As a user, I want to be able to register for an account | 1. Navigate to register page<br>2. Fill in required fields<br>3. Verify email<br>4. Log in with created credentials | User is able to register and log in with new account | PASS |
| As a user, I want to be able to log in and out of my account | 1. Navigate to login page<br>2. Enter credentials<br>3. Successfully log in<br>4. Navigate to logout<br>5. Confirm logout | User is able to log in and log out | PASS |
| As a user, I want to be able to recover my password | 1. Navigate to login page<br>2. Click forgot password<br>3. Enter email<br>4. Follow recovery link<br>5. Set new password | User receives email with recovery link and can set new password | PASS |
| As a user, I want to have a personalized profile | 1. Log in to account<br>2. Navigate to profile page<br>3. Update personal information<br>4. View order history | User can view and update profile information | PASS |

### Shopping Experience

| User Story | Test Procedure | Result | Pass/Fail |
|------------|---------------|--------|-----------|
| As a shopper, I want to view all products | 1. Navigate to all products page<br>2. Scroll through available products | All products are displayed correctly | PASS |
| As a shopper, I want to view individual product details | 1. Select a product<br>2. View detailed product information | Product details page shows all relevant information | PASS |
| As a shopper, I want to sort products by category | 1. Navigate to products page<br>2. Use category filter<br>3. Verify products shown match selected category | Products are correctly filtered by selected category | PASS |
| As a shopper, I want to search for products | 1. Use search bar<br>2. Enter search term<br>3. Verify search results | Search returns relevant products based on search term | PASS |
| As a shopper, I want to add products to my shopping bag | 1. View product details<br>2. Select quantity<br>3. Add to bag<br>4. Verify product appears in bag | Product is added to bag with correct quantity | PASS |

### Checkout & Payment

| User Story | Test Procedure | Result | Pass/Fail |
|------------|---------------|--------|-----------|
| As a shopper, I want to view items in my bag | 1. Add items to bag<br>2. Navigate to bag page<br>3. Verify items are displayed correctly | Bag shows all added items with correct details | PASS |
| As a shopper, I want to adjust quantities in my bag | 1. Navigate to bag<br>2. Update quantities<br>3. Verify changes are applied | Quantities can be adjusted and bag updates correctly | PASS |
| As a shopper, I want to enter payment information | 1. Proceed to checkout<br>2. Enter delivery information<br>3. Enter payment details<br>4. Complete order | Order is processed with entered payment details | PASS |
| As a shopper, I want to feel my payment is secure | 1. Verify secure payment icon<br>2. Check for https<br>3. Complete payment process | Payment system shows appropriate security indicators | PASS |
| As a shopper, I want to view order confirmation | 1. Complete checkout<br>2. Verify confirmation page appears<br>3. Check email for confirmation | Confirmation page displays and email is received | PASS |
